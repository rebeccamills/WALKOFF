How to Set up and Install WALKOFF

0) Make sure Python 2.7+ and git are installed.
1) Open a terminal on linux or a command prompt on Windows, and run the command "git clone https://github.com/iadgov/WALKOFF.git".
2) Change directories to the WALKOFF directory ("cd WALKOFF").
3) Run the command "python setup.py install" -- this command must be run as root (i.e. using sudo, or in an administrative command shell).
4) Run the command "pip install -r requirements.txt" to install the required libraries -- this command must be run as root (i.e. using sudo, or in an administrative command shell).
5) Each app must have a setup.py script inside of its main directory. In order to save the user the effort of manually running the setup.py script for each app, the user can run the command "python installDependencies.py -a|--apps AppOne AppTwo ..." -- this command must be run as root (i.e. using sudo, or in an administrative command shell). Simply provide the installDependencies script with the '-a' or '--apps' flag followed by a list of one or more apps, separated by spaces.
6) Run "python startServer.py" in your terminal or command prompt window.
7) Open a browser of your choice, and navigate to the host IP and port specified in the output of the above command to start the server. Typically, this will mean navigating to 127.0.0.1:5000, unless the user manually configured the host IP and port in the core.config file.
